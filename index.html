Data Exploration

Looking at the data, I‚Äôm interested in predicting ‚Äúanxiety score‚Äù from musical listening patterns with respect to genre frequencies. We‚Äôll need the anxiety and the frequency_*, so let‚Äôs visualize what we have here.

For the anxiety score interpretation on a scale from 0 to 10, respondents where asked ‚Äúhow often do you experience this?‚Äù:

  • 0 - I do not experience this.

  • 10 - I experience this regularly, constantly/or to an extreme.

  • Python

So, of no surprise to me, this data is right-skewed with a concentration sitting around a score from 7 - 8. The density curve also shows limited variability at lower anxiety levels (‚Äúflattening‚Äù of the curve), which may affect model performance in predicting across the full range of the outcome.

Data Cleaning & Preprocessing

  • R
  • Python

Now, we‚Äôll just grab the columns we need and make sure there‚Äôs no missing values or anything like that. In R, we can make our frequency_* variables into factors as well. Each frequency_* variable has four possible values: ‚ÄúNever‚Äù, ‚ÄúRarely‚Äù, ‚ÄúSometimes‚Äù, ‚ÄúVery frequently‚Äù to which each respondent can answer to the question ‚ÄúHow often do you listen to * genre?‚Äù

Code 
df_mh_music_r_sub <-
  df_mh_music_r |>
  # Only select what we need
  select(anxiety, starts_with("Frequency")) |>
  #
  drop_na() |>
  mutate(
    across(
      starts_with("Frequency"),
      \(x) {
        factor(
          x,
          levels = c("Never", "Rarely", "Sometimes", "Very frequently")
        )
      }
    )
  )

In python, there isn‚Äôt a direct ‚Äúfactor‚Äù equivalent like in R. We can make an equivalent while processing the data after we split it in the nest step. For now, I‚Äôll just select the relevant columns and make the frequency columns are casted as categorical just to be explicit.

Code 
import polars as pl
import polars.selectors as cs

# Select the relevant columns
freq_cols = df_mh_music_py.select(cs.starts_with("frequency")).columns
df_mh_music_py_sub = df_mh_music_py.select(["anxiety"] + freq_cols).drop_nulls()

# Casting frequency columns to categorical
df_mh_music_py_sub = df_mh_music_py_sub.with_columns(
    [pl.col(col).cast(pl.Categorical) for col in freq_cols]
)
